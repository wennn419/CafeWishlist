# CafeWishlist

## Description
Cafe Wishlist is a simple PHP and MySQL CRUD web application that allows users to save and manage cafes they want to visit.

## Features
- Add a cafe
- View all cafes
- Edit cafe information
- Delete a cafe
- Mark a cafe as visited

## Technologies
- PHP 8.2
- MySQL 8.0
- Docker
- Apache

## Run the project

```bash
docker compose up
```

Then open:

Application:
http://localhost:8080

phpMyAdmin:
http://localhost:8081
